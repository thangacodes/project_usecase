Amazon Machine Image include NGINX and other basic binaries installed by Packer.

What is Nginx? NGINX is open-source web server software used for reverse proxy, load balancing, and caching. It provides HTTPS server capabilities and is mainly designed for maximum performance and stability. It also functions as a proxy server for email communications protocols, such as IMAP, POP3, and SMTP.

What is the procedure for installing NGINX on an Amazon Linux Machine? Follow the instructions below.

  1. sudo yum update -y
  2. sudo yum install -y java
  3. sudo yum install -y nginx
  4. sudo systemctl enable nginx
  5. In order to change default html file in NGINX Server >> Please navigate to the dir /var/www/html
  6. Create custom index.html >> Move it to /var/www/html/
  7. After moving custom index.html file to the directory, restart NGINX service.
  8. sudo systemctl restart nginx

Custom Image contains the following:

  1. java jdk installed
  2. putty installed
  3. wget installed
  4. nginx web server installed
